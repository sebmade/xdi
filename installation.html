Notes d'installation

XDI nécessite d'utiliser des versions récentes de Xerces et Xalan, or celles-ci sont implémentées dans le JDK 1.4. Pour cela Sun fournit une technique pour supplanter l'implémentation intégrée dans le runtime : Endorsed Standards Override Mechanism.
Le principe consiste à placer un répertoire nommé endorsed dans le répertoire lib de son JRE. Dans ce répertoire il faut placer les jars des versions d'implémentations que vous voulez supplanter.
Pour XDI il faut placer dans le endorsed:

  • xercerImpl.jar version 2.6.2
  • xalan.jar version 2.6.0
  • xml-apis.jar dans Xerces 2.6.2
